## Polling App
- - -

#### Endpoints :

| URL        | HTTP Verb | Functions             |
|------------|-----------|-----------------------|
| 'user/login' | POST      | user login            |
| 'user/register' | POST      | user registration     |
| '/polls/{id}' | GET       | get specified ID poll |
| '/polls'   | GET       | get all polls         |

#### Tools :
1. Python
2. Django & Django Rest Framework
3. Postgres
